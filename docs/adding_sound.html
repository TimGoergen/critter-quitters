Adding Sound to Critter Quitters

Phase 6 — Sound infrastructure is wired. This document covers how to source audio files and drop them in.

How it works

All audio routes through AudioManager.gd, an autoloaded singleton at game/core/AudioManager.gd. Every sound slot is a variable initialised to null. Each play method checks for null first and returns silently if the slot is empty — so the game runs without error until real files are assigned.

Three audio buses are defined: Music, SFX, and UI, all routing to Master. Volume is controlled independently per bus. Music and SFX sliders appear in the right panel during play; UI is always at full volume.

No code changes needed anywhere else. Every hook (trap fires, enemy dies, wave starts, button pressed) is already wired. Adding a sound only requires dropping a file in the right folder and assigning one preload() line in AudioManager.gd.

Step-by-step: adding one sound

  1. Drop the file in the correct folder.
    Use .ogg for the music loop (smaller, streamed). Use .wav for all short SFX (no decode latency on play). See the folder map below.
  2. Set import options in Godot.
    For the music OGG: open the Import dock, tick Loop, click Re-import. For WAV files: defaults are fine — no changes needed.
  3. Assign the stream in AudioManager.gd.
    Replace the null for that slot with a preload(). See the pattern below.
  4. Test in-game.
    Run the game and trigger the event. If it doesn't play, check: file path matches exactly, import completed, variable name matches the slot.

Assignment pattern

Open game/core/AudioManager.gd. The stream slots are all near the top of the file. Replace any null with a preload(): 

# Before — slot is empty, sound plays silently
var sfx_trap_snap_fire: AudioStream = null

# After — file is assigned, sound plays on every Snap Trap fire
var sfx_trap_snap_fire: AudioStream = preload("res://assets/audio/sfx/traps/snap_fire.wav")

You can assign them one at a time. Every slot is independent — a missing file in one slot never affects any other sound.

File paths are case-sensitive on Android and Linux. The folder names in the tree above are all lowercase with underscores. Match them exactly.

Adding bus effects (optional)

The buses are created with no effects. You can add them at any time in the Godot editor without touching code:

  1. Open Project > Audio in the Godot editor.
  2. Click the Add Effect button on the Music bus. Add a Compressor to prevent the music from clipping over SFX.
  3. On the SFX bus, add an AudioEffectLimiter to prevent loud simultaneous hits from distorting.
  4. The UI bus needs no effects — it must always be audible and clean.
Bus effect settings are saved automatically to default_bus_layout.tres and committed with the project. You only need to do this once.

Enemy footstep sounds (6)

Played each time an enemy advances to a new grid cell. Each file is triggered repeatedly at a rate proportional to enemy speed, rate-limited to at most once per 0.1s per enemy instance. Files must be very short (<0.12s ideally) with no reverb tail — they loop rapidly and tails cause muddiness.

Variable File Enemy Character & search terms
sfx_step_ant sfx/enemies/step_ant.wav Ant (fast) Rapid light skitter — tiny tapping on a surface.
"ant footstep", "tiny scuttle tap", "insect light step"
sfx_step_gnat sfx/enemies/step_gnat.wav Gnat (very fast) Extremely light — near-silence, possibly a faint wing-buzz or tiny tap. Gnats are the smallest enemy; their audio presence should be minimal.
"tiny insect sound", "very quiet tap", "gnat buzz tiny"
sfx_step_cricket sfx/enemies/step_cricket.wav Cricket Chirpy skitter — slightly erratic feel to match the cricket's fast movement.
"cricket scuttle", "chirpy step insect", "light erratic skitter"
sfx_step_beetle sfx/enemies/step_beetle.wav Beetle (slow, heavy) Heavier click-clack — hard shell on a surface, slower rhythm to match the beetle's plodding movement.
"beetle footstep click", "hard shell step", "heavy insect walk"
sfx_step_cockroach sfx/enemies/step_cockroach.wav Cockroach Medium scrabbling — six-legged scramble, slightly skittery but with more weight than the ant or cricket.
"cockroach scuttle", "medium insect step", "scrabble skitter"
sfx_step_rat sfx/enemies/step_rat.wav Rat (boss, very slow) Padding thud — clearly heavier than any insect. Slow, deliberate footfall.
"rat footstep", "small animal heavy step", "padding thud soft"
Footstep tip: If a footstep file sounds too clean or long, trim it in Audacity or any audio editor to remove the tail. The goal is a single transient click or tap with no sustain.
